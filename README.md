## About

Frontend client for the Agenda task management system. Built with Next.js and TypeScript for a modern, responsive user experience.

## Features

- 📋 **Task Dashboard** — Overview of all tasks
- ✏️ **Inline Editing** — Edit tasks without page reload
- 🎨 **Modern UI** — Clean design with Tailwind CSS
- 📱 **Responsive** — Works on all screen sizes

## Getting Started

```bash
git clone https://github.com/foxnaim/Agenda-client-.git
cd Agenda-client-
yarn install
yarn dev
```
